# 🎮 RetroMini Games

A collection of fun mini-games with built-in screen-time management. Set a timer, play any of 8+ classic games, and the studio automatically closes when time's up. 

## Features

- **8+ Mini-Games**: Number Quest, Reaction Blitz, Word Quiz, Tic Tac Toe, Snake, Tetris, Flappy Bird, Minesweeper
- **Screen-Time Manager**: Set custom timers with presets and automatic studio closure
- **Stats Tracking**: Games played, wins, best reaction time, and session history
- **Modern Design**: Clean, responsive UI with smooth animations
- **Zero Dependencies**: Pure HTML, CSS & JavaScript

## How to Play

1. Open `retromini-games.html` in your browser
2. Set your screen-time limit using the Timer section
3. Browse and select a game
4. Play while the timer counts down
5. Stats track your performance automatically

## Technologies

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 with CSS Variables
- Responsive Web Design
